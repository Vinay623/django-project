# Campus Enrolment Manager

## Description
This is a Django web application to manage students, courses, and enrolments.

## Features
- Add Students
- Add Courses
- Manage Enrolments

## Technologies Used
- Python
- Django

## How to Run
1. pip install -r requirements.txt
2. python manage.py migrate
3. python manage.py runserver
